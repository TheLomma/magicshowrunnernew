## GitHub-Setup

1. Ordner `icons/` im Repo anlegen
2. Alle PNG-Dateien in `icons/` hochladen
3. `manifest.json` ins Root-Verzeichnis
4. `head_snippet.html`-Inhalt in den `<head>` deiner `index.html` kopieren

## iPhone Home Screen

1. Safari öffnen → deine GitHub Pages URL
2. Teilen-Symbol → „Zum Home-Bildschirm"
3. Das Icon erscheint automatisch ✅
